## Обзор экранов

```
[Заставка] → [Главное меню] → [Выбор сложности] → [Карта акта] → [Бой/Событие/...] → [Результат рана]
                  ↓                                       ↓
            [Настройки]                              [Магазин]
                  ↓                                       ↓
            [Туториал]                             [Место отдыха]
```

## Экран 1: Заставка (Splash Screen)

**Путь:** Запуск игры → автопоказ 2-3 сек → Главное меню

- Логотип "Lexica Spire"
- Анимация: книга раскрывается, буквы разлетаются
- Пропуск: клик/любая клавиша

## Экран 2: Главное меню

**Элементы UI:**
```
┌────────────────────────────┐
│      LEXICA SPIRE          │ ← Готический шрифт, Merriweather
│                            │
│  [▶ Новый ран]             │
│  [📂 Продолжить]           │ ← Серая если нет сейва
│  [📊 Статистика]           │
│  [⚙️ Настройки]            │
│  [📘 Туториал]             │
│  [❌ Выход]                │
└────────────────────────────┘
```

**Действия пользователя:**

| Кнопка | Переход | Условие |
|--------|---------|---------|
| Новый ран | → Экран выбора сложности | Всегда |
| Продолжить | → Карта акта (загрузка сейва) | Есть сохранение |
| Статистика | → Экран статистики (лучший ран, слова, точность) | Всегда |
| Настройки | → Экран настроек | Всегда |
| Туториал | → Обучающая последовательность (3 нода) | Всегда |
| Выход | Закрытие приложения | Всегда |

## Экран 3: Выбор сложности (CEFR Level)

**Путь:** Главное меню → Новый ран → Выбор сложности → Карта акта

```
┌─────────────────────────────────────┐
│  Выберите уровень сложности:        │
│                                     │
│  [A1 - Beginner]     Простые слова  │
│  [A2 - Elementary]   Базовые фразы  │
│  [B1 - Intermediate] Средний        │ ← По умолчанию
│  [B2 - Upper]        Продвинутый    │
│  [C1 - Advanced]     Сложный        │
│                                     │
│  [Начать ран →]                     │
└─────────────────────────────────────┘
```

**Логика:**
1. Игрок выбирает CEFR уровень → определяет пул слов и сложность заданий
2. Нажимает "Начать ран"
3. Генерируется стартовая колода (10 карт) + карта Акта 1
4. → Переход на Карту акта

## Экран 4: Карта акта (Act Map)

**Путь:** Выбор сложности → Карта акта ↔ (бои, события, магазин, отдых)

```
         👑 БОСС: Ашипу
          |
    ⚔️──┬──❓──┬──💰         Уровень 15
       |        |
    📚──┼──🔥──┼──⚔️        Уровни 10-14
       |        |
    👿──┼──❓──┼──📚         Уровни 6-9
       |        |
    ⚔️──┼──💰──┼──⚔️        Уровни 2-5
       |        |
    ⚔️──┴──⚔️──┴──⚔️        Уровень 1
          |
         🎯 СТАРТ
```

**Типы нодов:**

| Иконка | Тип | Действие при клике |
|--------|-----|-------------------|
| ⚔️ | Бой | → Экран боя (обычный враг) |
| 👿 | Элитный бой | → Экран боя (усиленный враг, лучшие награды) |
| 📚 | Библиотека | → Экран выбора карт (3 карты на выбор) |
| 🔥 | Место отдыха | → Экран отдыха (лечение ИЛИ учёба) |
| 💰 | Торговец | → Экран магазина |
| ❓ | Событие | → Экран случайного события |
| 👑 | Босс | → Экран боя (босс акта) |

**Поведение:**
- Доступны только ноды, соединённые с текущей позицией
- Пройденные ноды затемнены + галочка
- Hover на ноде → подсказка (тип врага / награда)
- Информационная панель: HP игрока, Lex-Coins, количество карт/проклятий

## Экран 5: Бой (Combat Screen)

**Путь:** Карта → Нод боя → Бой → Награды → Карта

### Лэйаут боя
```
┌─────────────────────────────────────────────────────────┐
│  HP: ████████░░ 78/100     Energy: ⚡⚡⚡  Lex: 💰 245   │
│  Реликвии: [📖][⚔️][🎓]   Проклятия: ⛓️×2              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│            [Спрайт врага]                               │
│            УТГАЛУ                                       │
│            HP: ███████░░░ 28/40                         │
│            Намерение: 🗡️ 7 урона + 🌀 Confusion         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  [Карта 1] [Карта 2] [Карта 3] [Карта 4] [Карта 5]     │
│   1⚡ Atk    1⚡ Def    2⚡ Atk    1⚡ Def    ⛓️ Curse    │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  Колода: 12  │  Сброс: 5  │  Исчерпано: 0  │ [Конец хода] │
└─────────────────────────────────────────────────────────┘
```

### Пошаговый flow одного хода

```
1. Начало хода
   ├─ Восстановить Energy до 3
   ├─ Взять 5 карт (минус Fatigue стаки)
   └─ Показать намерение врага

2. Фаза игрока (повторяется пока есть Energy)
   ├─ Игрок кликает карту в руке
   ├─ Проверка Energy (хватает?)
   │   ├─ Нет → подсветить карту красным, ничего не происходит
   │   └─ Да → открыть окно задания
   │
   ├─ ЗАДАНИЕ (центр экрана, поверх боя)
   │   ┌──────────────────────────────┐
   │   │  Что означает "ubiquitous"?  │
   │   │                              │
   │   │  A) Rare and valuable        │
   │   │  B) Found everywhere   ✓     │
   │   │  C) Dangerous                │
   │   │  D) Ancient                  │
   │   │                              │
   │   │  Таймер: ████████░░ 7s       │
   │   └──────────────────────────────┘
   │
   ├─ Оценка ответа
   │   ├─ Perfect (< 2 сек): +25% эффект, +1 Insight
   │   ├─ Correct (2-10 сек): 100% эффект
   │   ├─ Slow (10-20 сек): 75% эффект, +1 Fatigue
   │   └─ Mistake: карта fizzle, -5 HP, +1 Curse в колоду
   │
   ├─ Применение эффекта карты
   │   ├─ Attack → анимация урона, число над врагом
   │   ├─ Skill → анимация блока, щит на игроке
   │   └─ Power → постоянный баф, иконка в панели
   │
   └─ Карта уходит в сброс

3. Игрок нажимает "Конец хода"
   ├─ Оставшиеся карты → сброс
   ├─ Ход врага
   │   ├─ Атака → урон игроку (блок поглощает)
   │   ├─ Дебаф → наложить эффект (Confusion/Silence/etc)
   │   └─ Баф → враг усиливается
   ├─ Блок игрока сбрасывается
   └─ → Начало следующего хода
```

## Экран 6: Задание — Grammar (текстовый ввод)

```
┌────────────────────────────────────┐
│  Исправьте предложение:            │
│                                    │
│  "She don't like apples."          │
│                                    │
│  Ваш ответ:                       │
│  [She doesn't like apples.     ]   │
│                                    │
│  [Отправить]     [Пропустить]      │
├────────────────────────────────────┤
│  Таймер: ██████████ 10s            │
└────────────────────────────────────┘
```

**Flow:**
1. Отображается ошибочное предложение
2. Игрок вводит исправленную версию
3. Нажимает "Отправить" (или Enter)
4. Backend валидирует → Perfect / Correct / Slow / Mistake
5. Feedback: зелёная/красная подсветка + правильный ответ при ошибке

## Экран 14: Туториал (Tutorial)

**Путь:** Главное меню → Туториал → 3 обучающих боя → Главное меню

### Шаг 1: Основы карт
- Показать руку из 3 простых карт
- Подсветить карту → "Нажмите на карту, чтобы сыграть"
- Показать окно задания → "Ответьте правильно, чтобы активировать карту"

### Шаг 2: Типы ответов
- Дать задание с гарантированным Perfect → объяснить бонус
- Дать задание с таймером → показать как работает Slow penalty
- Намеренно показать Mistake → объяснить проклятия

### Шаг 3: Враг и стратегия
- Показать намерение врага → "Он атакует на 7. Используйте Skill карту для блока"
- Объяснить End Turn → ход врага
- Победить врага → показать награды

## Навигационная карта (полная)

```
Заставка
  └→ Главное меню
       ├→ Новый ран → Выбор сложности → Карта акта ─┐
       ├→ Продолжить ─────────────────→ Карта акта ─┤
       ├→ Статистика                                 │
       ├→ Настройки                                  │
       ├→ Туториал → Обучение (3 боя) → Главное меню│
       └→ Выход                                      │
                                                     │
       Карта акта ←──────────────────────────────────┘
         ├→ Бой → Награды → Карта акта
         ├→ Элитный бой → Награды (+ реликвия) → Карта акта
         ├→ Библиотека → Выбор карты → Карта акта
         ├→ Место отдыха → (Отдых | Учёба) → Карта акта
         ├→ Магазин → Покупки → Карта акта
         ├→ Событие → Выбор опции → Последствия → Карта акта
         └→ Босс → Бой с боссом ─┐
                                  ├→ Победа → Награды → Результат рана → Главное меню
                                  └→ Поражение → Результат рана → Главное меню
```
