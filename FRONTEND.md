# FRONTEND — Lexica Spire (Визуал и стили)

**Версия:** 1.0
**Дата:** 2026-02-13

---

## Визуальный стиль

**Концепция:** *Academic Gothic meets Roguelike Dungeon*

Сочетание академической эстетики (пергамент, чернила, золотые акценты) с мрачной атмосферой подземелья. Вдохновение: Slay the Spire (рисованный стиль), Darkest Dungeon (контраст, драма), Inscryption (оккультная академичность).

**Графика:** AI-генерация с ручной доработкой для единого стиля.

## Компоненты UI

### Карта (Card Component)

```
┌───────────────────────┐  Размер: 180 × 260 px
│ ⚡1    ATTACK     🗡️  │  Верх: стоимость + тип + иконка
├───────────────────────┤
│                       │
│    [ИЛЛЮСТРАЦИЯ]      │  Центр: AI-арт (140 × 140 px)
│                       │
├───────────────────────┤
│  Vocabulary Strike    │  Название: Merriweather Bold 16px
├───────────────────────┤
│  Deal 8 damage.       │  Эффект: Merriweather 14px
│  Challenge: Vocab     │  Тип задания: иконка + текст
└───────────────────────┘
```

**Состояния карты:**

| Состояние | Визуал |
|-----------|--------|
| В руке (idle) | Обычный вид, лёгкая тень |
| Hover | Подъём на 20px, увеличение 1.1x, яркая тень |
| Выбрана (dragging) | Полупрозрачная, следует за курсором |
| Не хватает Energy | Затемнение 50%, красная подсветка стоимости |
| Проклятие | Зелёное свечение `#6B8E23`, трещины на арте |
| Улучшенная (+) | Зелёное свечение контура, "+" в названии |

**Рамки по типу:**
- Attack: рамка `#8B1E1E` (2px)
- Skill: рамка `#2E5090` (2px)
- Power: рамка `#2D5F3F` (2px)
- Curse: рамка `#5A3E5C` (2px, неровная/потрескавшаяся)

## Анимации

### Быстрые (< 0.3 сек)
- Взятие карты: slide from deck + ease-out
- Числа урона: pop-up float + fade
- Иконки статус-эффектов: fade-in

### Средние (0.3-0.7 сек)
- Игра карты: slide to center → challenge popup
- Атака врага: shake + flash
- Получение блока: щит shimmer

### Долгие / Пропускаемые (> 1 сек)
- Появление босса: cutscene (Skip: клик)
- Получение реликвии: fanfare + glow
- Переход между актами

### Easing функции
- UI элементы: `ease_out_cubic`
- Карты: `ease_out_back` (лёгкий "отскок")
- Урон: `ease_in_out_sine`

## Accessibility

| Функция | Реализация |
|---------|------------|
| Дальтонизм | Иконки + текстовые метки для всех цветовых индикаторов |
| Дислексия | Переключение на OpenDyslexic в настройках |
| Масштаб текста | 100% / 125% / 150% |
| Контраст | Все тексты проходят WCAG AA (contrast ratio > 4.5:1) |
| Тактильная обратная связь | Тряска экрана при ошибке (отключаемо) |
